Welcome to Bacdiving's documentation!
=====================================
**Bacdiving** is a Python package which can access and retrieve information from the world's largest database for standardized bacterial phenotypic information: `BacDive <https://bacdive.dsmz.de/>`_. 
Additionally, Bacdiving provides access statistics and several options to visualize the retrieved this information.


Check out the :doc:`installation` section on how to install this package as well as the :doc:`usage` and :doc:`tutorial` section for further information on how to use the package.
